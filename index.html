Twitter Quotter

Twitter quotter is a bot made using python that scrapes the internet for the famous quotes and tweets it along with the predefined hashtags for a particular day. Also if you tag the bot and use '#qod' in your tweet, it will reply you back with a generated image of a famous quote. And also it will not forget to like your tweet too.

JoBot

Jobot is a ChatBot Service for android made using Java. It is integrated with PayPal Payments api and secured with Firebase authentication. The login and signup to the app are handled by Firebase Authentication along with all the necessary functions of Forgot Password , number of failed login attempts etc. The paypal payments api is used for donation links. This Chatbot can tell you jokes along with having some nice small conversations, hence can relax your mind!. The Chatbot was designed and integrated in the app using dialogflow.

EverNotes

EverNotes is and online platform for sharing notes among teachers and students. All the users and the notes can managed by the admin of the particular department. All the uploaded notes will have the details like the name of the person who uploaded,time,type of file etc. This platform was build using PHP during my 2 months of internship at NIIT Ltd.(S19) I also increased the efficiency of the website by using image compression and used buffering and caching to decrease the server load and access time.

Bike Rental Demand Predection

Bike sharing systems are a means of renting bicycles where the process of obtaining membership, rental, and bike return is automated via a network of kiosk locations throughout a city. I Built a linear regression and a decision tree model to combine historical usage patterns with weather data in order to forecast hourly bike rental demand. RMSLE was considered as the evaluation metric which comes out to be 0.051 for the decision tree model.

Youtube Automation

Created a python script to automatically update the title of a youtube video as the number of views increases in real time with youtube. Also it updates the description showing the exact number of likes the video has at that point of time. It first fetches the statistics of all the videos on a channel and by passing the video id only, it can update the video's meta data with the latest fetched details.
